\doxysubsection{Detailed Description}
Singleton-\/based handler for system signals. 

The \mbox{\hyperlink{classSignalHandler}{Signal\+Handler}} class provides methods to set up handlers for system signals, particularly for enabling graceful shutdown mechanisms. Being designed as a singleton ensures that only one instance handles all signals in a consistent manner.
